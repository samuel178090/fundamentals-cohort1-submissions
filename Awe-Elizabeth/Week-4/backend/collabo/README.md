### Collabo-Backend

## Getting Started
1. Clone repository: https://github.com/Awe-Elizabeth/Collabo-backend.git
2. Run npm install
3. Populate a .env file using the sample below
4. Run app with "npm run dev"

# Sample ENV
`MONGOURL=mongodb+srv://<username>:<password>@cluster0.gtqhq.mongodb.net/<dbName>?retryWrites=true&w=majority&appName=Cluster0
PORT=3037
JWT_SECRET=supersecretencryptonkey
NODE_ENV=development
ENC_KEY=supersecretencryptonkey
IV=supersecretencryptonkey

`

## Frontend Link
1. Clone repository: https://github.com/Awe-Elizabeth/Collabo-frontend.git
2. Run "npm install"
3. Run app with "npm run dev"

## Post flow
Users are expected to register and then create new project posts. Their own posts can be viewed and they can view the post of others to collaborate

## Comment Flow:
Users are able to add comments to the project posts of others.
