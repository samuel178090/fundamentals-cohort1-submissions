# Cart Service Frontend

This is the **Cart Service frontend** built with **React and Vite**.  
It communicates with the Cart Service backend to display and manage a user's shopping cart.

---

The Backend repo => "https://github.com/IsamaDes/cart-service-api"

## **Features**

- Display items in a user's cart
- Add items to the cart via a form
- Simple routing with React Router
- Built with TypeScript-ready setup (TSX compatible)

---

## **Tech Stack**

- React 18
- Vite
- React Router DOM
- TypeScript-ready (optional)

---

## **Getting Started**

### **Prerequisites**

- Node.js >= 18
- npm
- Cart Service API running at `http://localhost:5000`

### **Installation**

1. Clone the repo:

```bash
git clone https://github.com/yourusername/cart-service-fe.git
cd cart-service-fe
```
